Multispectral Camera Crop Health Detection

//////////////////////////////////////////////

Overview

This project captures multispectral images using RGB and NIR cameras to analyze crop health. The script calculates NDVI (Normalized Difference Vegetation Index) to assess vegetation condition and visualize plant stress levels.

-------------------------------------------------------------------
Features
----------------
Captures images from RGB & NIR cameras

Extracts Red and NIR channels for analysis

Computes NDVI values to indicate crop health

Displays an NDVI heatmap (Green = healthy, Red = stressed)

-------------------------------------------------------------------
Applications
---------------
Precision farming

Crop health monitoring

Drought and irrigation assessment

Early disease detection
